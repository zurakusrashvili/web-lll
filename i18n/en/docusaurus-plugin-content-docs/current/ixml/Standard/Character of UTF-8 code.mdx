# Character of UTF-8 code

### Anatomy

```xml
<char var="var">
    int
</char>
```

 **Description**:
 CHAR returns the specific character of an UTF-8 code.

### Content: int

```xml
<content type="int"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | char |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var |  | N/A |
