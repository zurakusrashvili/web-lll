# Random array item

### Anatomy

```xml
<array:rand var="var" var_key="var" var_value="var"/>
```

 **Description**:
 ARRAY:RAND returns a random item of an array.

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Variable name | array:rand |
| var_key | var | Result variable name for key | array:rand |
| var_value | var | Result variable name for value | array:rand |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var_key |  | no-result-propagation |
| var_value |  | N/A |
