# Hyperbolic sine

### Anatomy

```xml
<math:sinh var="var">
    float
</math:sinh>
```

 **Description**:
 MATH:SINH calculates the hyperbolic sine of a value.

### Content: float

```xml
<content type="float"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | math:sinh |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | float | N/A |


## Examples

### Example: Example

```xml
<output>
<math:sinh>1</math:sinh>
</output>
<!-- 1.1752011936438 -->
```
