# Template

### Anatomy

```xml
<pdf:template source="int" target="int"/>
```

**Parents**: pdf:templates

 **Description**:
 PDF:TEMPLATE imports a page as a template.

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| source | int | Source page number | pdf:template |
| target | int | Target page number | pdf:template |
