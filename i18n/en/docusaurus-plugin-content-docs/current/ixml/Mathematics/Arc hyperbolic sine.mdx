# Arc hyperbolic sine

### Anatomy

```xml
<math:asinh var="var">
    float
</math:asinh>
```

 **Description**:
 MATH:ASINH calculates the inverse hyperbolic sine of a value.

### Content: float

```xml
<content type="float"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | math:asinh |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | float | N/A |


## Examples

### Example: Example

```xml
<output>
	<math:asinh>1</math:asinh>
</output>

<!-- 0.88137358701954 -->
```
