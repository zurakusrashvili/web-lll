# Connection

### Anatomy

```xml
<db:connection server="string" dbname="string" username="string" password="string" type="type">
    ixml
</db:connection>
```

 **Description**:
 DB:CONNECTION establishes a database connection.

### Content: ixml

```xml
<content type="ixml"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| server | string | Server | db:connection |
| dbname | string | Database name | db:connection |
| username | string | Username | db:connection |
| password | string | Password | db:connection |
| type | type | Database type | db:connection |
