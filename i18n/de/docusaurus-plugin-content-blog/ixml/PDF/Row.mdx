# Row

### Anatomy

```xml
<pdf:row leftmargin="float" rightmargin="float" x="float" y="float">
    ixml
</pdf:row>
```

**Parents**: pdf:header, pdf:footer, pdf:body

 **Description**:
 PDF:ROW prints a row of text columns as tabular content.

### Content: ixml

```xml
<content type="ixml"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| leftmargin | float | Left margin | pdf:row |
| rightmargin | float | Right margin | pdf:row |
| x | float | Abscissa | pdf:row |
| y | float | Ordinate | pdf:row |
