# Exponent of Euler's constant

### Anatomy

```xml
<math:exp var="var">
    float
</math:exp>
```

 **Description**:
 MATH:EXP raises Euler's constant (e) to a given power.

### Content: float

```xml
<content type="float"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | math:exp |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | float | N/A |


## Examples

### Example: Example

```xml
<output>
	<math:exp>10</math:exp>
</output>
```
