# List entries

### Anatomy

```xml
<zip:list var="var"/>
```

**Parents**: zip:archive

 **Description**:
 ZIP:LIST lists the filenames of all entries within an archive.

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | zip:list |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | array | no-result-propagation |
