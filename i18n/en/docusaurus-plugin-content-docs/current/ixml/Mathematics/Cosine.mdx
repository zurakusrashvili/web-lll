# Cosine

### Anatomy

```xml
<math:cos var="var">
    float
</math:cos>
```

 **Description**:
 MATH:COS calculates the cosine of a value.

### Content: float

```xml
<content type="float"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | math:cos |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | float | N/A |


## Examples

### Example: Example

```xml
<output>
	<math:cos>1</math:cos>
</output>

<!-- 0.54030230586814 -->
```
