# Add row

### Anatomy

```xml
<excel:addrow index="int" height="float"/>
```

**Parents**: excel:workbook

 **Description**:
 EXCEL:ADDROW adds a new row.

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| index | int | Row index | excel:addrow |
| height | float | Height | excel:addrow |
