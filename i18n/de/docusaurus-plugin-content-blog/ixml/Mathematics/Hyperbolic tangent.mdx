# Hyperbolic tangent

### Anatomy

```xml
<math:tanh var="var">
    float
</math:tanh>
```

 **Description**:
 MATH:TANH calculates the hyperbolic tangent of a value.

### Content: float

```xml
<content type="float"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | math:tanh |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | float | N/A |


## Examples

### Example: Example

```xml
<output>
<math:tanh>1</math:tanh>
</output>
<!-- 0.76159415595576 -->
```
