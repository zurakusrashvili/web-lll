# Arc hyperbolic cosine

### Anatomy

```xml
<math:acosh var="var">
    float
</math:acosh>
```

 **Description**:
 MATH:ACOSH calculates the inverse hyperbolic cosine of a value.

### Content: float

```xml
<content type="float"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | math:acosh |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | float | N/A |
