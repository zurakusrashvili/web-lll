# Set position

### Anatomy

```xml
<pdf:setpos page="int" x="float" y="float"/>
```

**Parents**: pdf:header, pdf:footer, pdf:body

 **Description**:
 PDF:SETPOS sets the position.

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| page | int | Page number | pdf:setpos |
| x | float | Abscissa | pdf:setpos |
| y | float | Ordinate | pdf:setpos |
