# Arc hyperbolic tangent

### Anatomy

```xml
<math:atanh var="var">
    float
</math:atanh>
```

 **Description**:
 MATH:ATANH calculates the inverse hyperbolic tangent of a value.

### Content: float

```xml
<content type="float"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | math:atanh |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | float | N/A |


## Examples

### Example: Example

```xml
<output>
<math:atanh>0</math:atanh>
</output>
<!-- 0 -->
```
