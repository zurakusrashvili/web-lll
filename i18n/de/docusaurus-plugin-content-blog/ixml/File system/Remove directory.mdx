# Remove directory

### Anatomy

```xml
<file:removedir dirname="string"/>
```

 **Description**:
 FILE:REMOVEDIR removes an empty directory.

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| dirname | string | Directory name/URL | file:removedir |


## Examples

### Example: Example

```xml
<file:removedir dirname="./test/ixml_dir"/>
```
