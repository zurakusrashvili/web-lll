# Rename entry

### Anatomy

```xml
<zip:rename oldname="string" newname="string"/>
```

**Parents**: zip:archive

 **Description**:
 ZIP:RENAME renames an entry within an archive from an old filename to a new filename.

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| oldname | string | Old filename | zip:rename |
| newname | string | New filename | zip:rename |
