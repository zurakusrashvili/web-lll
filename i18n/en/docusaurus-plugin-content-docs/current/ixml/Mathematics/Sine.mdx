# Sine

### Anatomy

```xml
<math:sin var="var">
    float
</math:sin>
```

 **Description**:
 MATH:SIN calculates the sine of a value.

### Content: float

```xml
<content type="float"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | math:sin |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | float | N/A |


## Examples

### Example: Example

```xml
<output>
<math:sin>1</math:sin>
</output>
<!-- 0.8414709848079 -->
```
