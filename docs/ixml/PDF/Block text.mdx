# Block text

### Anatomy

```xml
<pdf:block html="bool" nowrap="bool" height="float" width="float" leftmargin="float" rightmargin="float" x="float" y="float">
    string
</pdf:block>
```

**Parents**: pdf:header, pdf:footer, pdf:body

 **Description**:
 PDF:BLOCK prints a block of text as sectioning content.

### Content: string

```xml
<content type="string"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| html | bool | HTML content | pdf:block |
| nowrap | bool | No text wrapping | pdf:block |
| height | float | Minimum Height | pdf:block |
| width | float | Width | pdf:block |
| leftmargin | float | Left margin | pdf:block |
| rightmargin | float | Right margin | pdf:block |
| x | float | Abscissa | pdf:block |
| y | float | Ordinate | pdf:block |
