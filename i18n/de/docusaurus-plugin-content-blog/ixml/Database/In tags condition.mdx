# In tags condition (ZeyOS)

### Anatomy

```xml
<db:intags var="var" entity="string" field="string"/>
```

**Parents**: db:lookup, db:select, db:inner, db:innerextdata, db:innertags, db:left, db:leftextdata, db:lefttags, db:having, db:update, db:delete

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Variable name for static tag names list | db:intags |
| entity | string | Entity name | db:intags |
| field | string | Field name | db:intags |
