# Outer left join on tags (ZeyOS)

### Anatomy

```xml
<db:lefttags alias="string" entity="string" name="string" field="string"/>
```

**Parents**: db:join

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| alias | string | Alias name | db:lefttags |
| entity | string | Entity name | db:lefttags |
| name | string | Tag name | db:lefttags |
| field | string | Field name | db:lefttags |
