# Square root

### Anatomy

```xml
<math:sqrt var="var">
    float
</math:sqrt>
```

 **Description**:
 MATH:SQRT calculates the square root of a value.

### Content: float

```xml
<content type="float"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | math:sqrt |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | float | N/A |


## Examples

### Example: Example

```xml
<output>
	<math:sqrt>25</math:sqrt>
</output>

<!-- 5 -->
```
