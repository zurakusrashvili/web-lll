# Add sheet

### Anatomy

```xml
<excel:addsheet index="int">
    string
</excel:addsheet>
```

**Parents**: excel:workbook

 **Description**:
 EXCEL:ADDSHEET adds a new sheet.

### Content: string

```xml
<content type="string"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| index | int | Sheet index | excel:addsheet |
