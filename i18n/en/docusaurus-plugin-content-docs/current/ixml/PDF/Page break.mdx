# Page break

### Anatomy

```xml
<pdf:pagebreak offset="float"/>
```

**Parents**: pdf:header, pdf:footer, pdf:body

 **Description**:
 PDF:PAGEBREAK adds a new page and shifts the ordinate to a given offset.

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| offset | float | Offset | pdf:pagebreak |
