# Natural logarithm

### Anatomy

```xml
<math:ln var="var">
    float
</math:ln>
```

 **Description**:
 MATH:LN calculates the natural logarithm of a value.

### Content: float

```xml
<content type="float"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | math:ln |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | float | N/A |


## Examples

### Example: Example

```xml
<output>
<math:ln>22026.465794807</math:ln>
</output>
<!-- 10 -->
```
