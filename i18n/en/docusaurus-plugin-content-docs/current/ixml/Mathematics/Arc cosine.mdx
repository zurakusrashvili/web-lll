# Arc cosine

### Anatomy

```xml
<math:acos var="var">
    float
</math:acos>
```

 **Description**:
 MATH:ACOS calculates the inverse cosine of a value.

### Content: float

```xml
<content type="float"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | math:acos |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
