# Arc sine

### Anatomy

```xml
<math:asin var="var">
    float
</math:asin>
```

 **Description**:
 MATH:ASIN calculates the inverse sine of a value.

### Content: float

```xml
<content type="float"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | math:asin |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | float | N/A |


## Examples

### Example: Example

```xml
<output>
	<math:asin>1</math:asin>
</output>

<!-- 1.5707963267949 -->
```
