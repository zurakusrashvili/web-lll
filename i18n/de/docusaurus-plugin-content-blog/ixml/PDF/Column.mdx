# Column

### Anatomy

```xml
<pdf:col html="bool" nowrap="bool" height="float" width="float">
    string
</pdf:col>
```

**Parents**: pdf:row

### Content: string

```xml
<content type="string"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| html | bool | HTML content | pdf:col |
| nowrap | bool | No text wrapping | pdf:col |
| height | float | Minimum Height | pdf:col |
| width | float | Width | pdf:col |
