# Document

### Anatomy

```xml
<pdf:document var="var" author="string" title="string" subject="string" keywords="string" subsetting="bool" unit="string" zoom="string" layout="string" mode="string">
    ixml
    <pdf:signature signcert="string" privatekey="string" password="string" extracerts="string"/>
</pdf:document>
```

 **Description**:
 PDF:DOCUMENT generates a PDF document that is comprised of multiple sections.

### Content: ixml

```xml
<content type="ixml"/>
```
