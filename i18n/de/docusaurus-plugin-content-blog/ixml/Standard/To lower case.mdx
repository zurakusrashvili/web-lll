# To lower case

### Anatomy

```xml
<tolower var="var">
    string
</tolower>
```

 **Description**:
 TOLOWER returns the lower case representation of a value.

### Content: string

```xml
<content type="string"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | tolower |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | string | N/A |


## Examples

### Example: Example

```xml
<output>
<tolower>iXML</tolower>
</output>
<!-- ixml -->
```
