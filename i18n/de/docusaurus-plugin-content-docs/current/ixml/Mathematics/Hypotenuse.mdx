# Hypotenuse

### Anatomy

```xml
<math:hypot var="var" x="float" y="float"/>
```

 **Description**:
 MATH:HYPOT calculates the length of the hypotenuse of a right-angle triangle.

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | math:hypot |
| x | float | Abscissa | math:hypot |
| y | float | Ordinate | math:hypot |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | float | N/A |


## Examples

### Example: Example

```xml
<output>
<math:hypot x="3" y="5"/>
</output>
<!-- 5.8309518948453 -->
```
