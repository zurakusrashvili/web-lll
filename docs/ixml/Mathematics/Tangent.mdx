# Tangent

### Anatomy

```xml
<math:tan var="var">
    float
</math:tan>
```

 **Description**:
 MATH:TAN calculates the tangent of a value.

### Content: float

```xml
<content type="float"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | math:tan |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | float | N/A |


## Examples

### Example: Example

```xml
<output>
	<math:tan>1</math:tan>
</output>

<!-- 1.5574077246549 -->
```
