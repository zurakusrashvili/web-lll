# Insert record

### Anatomy

```xml
<db:insert var="var" var_data="var" table="string">
    ixml
</db:insert>
```

**Parents**: db:connection

 **Description**:
 DB:INSERT executes a SQL/DML insert statement that inserts a new row into a table within the database.

### Content: ixml

```xml
<content type="ixml"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name for insert ID | db:insert |
| var_data | var | Variable name for associated field name and value pairs | db:insert |
| table | string | Table name | db:insert |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | int | N/A |
