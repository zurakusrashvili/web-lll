# Data

### Anatomy

```xml
<db:data field="string" alias="string">
    string
</db:data>
```

**Parents**: db:get, db:set, db:fields, db:insert, db:update

### Content: string

```xml
<content type="string"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| field | string | Field name | db:data |
| alias | string | Alias name | db:data |
