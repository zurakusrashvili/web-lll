# Arc tangent

### Anatomy

```xml
<math:atan var="var">
    float
</math:atan>
```

 **Description**:
 MATH:ATAN calculates the inverse tangent of a value.

### Content: float

```xml
<content type="float"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | math:atan |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | float | N/A |


## Examples

### Example: Example

```xml
<output>
<math:atan>1</math:atan>
</output>
<!-- 0.78539816339745 -->
```
