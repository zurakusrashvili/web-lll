# Add column

### Anatomy

```xml
<excel:addcol index="int" width="float">
    string
</excel:addcol>
```

**Parents**: excel:workbook

 **Description**:
 EXCEL:ADDCOL adds a new column.

### Content: string

```xml
<content type="string"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| index | int | Column index | excel:addcol |
| width | float | Width | excel:addcol |
