# Hyperbolic cosine

### Anatomy

```xml
<math:cosh var="var">
    float
</math:cosh>
```

 **Description**:
 MATH:COSH calculates the hyperbolic cosine of a value.

### Content: float

```xml
<content type="float"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | math:cosh |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | float | N/A |


## Examples

### Example: Example

```xml
<output>
<math:cosh>1</math:cosh>
</output>
<!-- 1.5430806348152 -->
```
