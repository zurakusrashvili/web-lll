# Set cell value

### Anatomy

```xml
<excel:set coords="string" format="string" type="type">
    string
</excel:set>
```

**Parents**: excel:workbook

 **Description**:
 EXCEL:SET sets the cell value at the current position of the cell cursor or at given coordinates.

### Content: string

```xml
<content type="string"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| coords | string | Cell coordinates | excel:set |
| format | string | Format | excel:set |
| type | type | Data type | excel:set |
