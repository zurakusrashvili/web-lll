# Barcode

### Anatomy

```xml
<pdf:barcode type="string" height="float" width="float" x="float" y="float" resolution="float">
    string
</pdf:barcode>
```

**Parents**: pdf:header, pdf:footer, pdf:body

 **Description**:
 PDF:BARCODE prints a linear or matrix bar code, specified by a given type.

### Content: string

```xml
<content type="string"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| type | string | Barcode type | pdf:barcode |
| height | float | Height | pdf:barcode |
| width | float | Width | pdf:barcode |
| x | float | Abscissa | pdf:barcode |
| y | float | Ordinate | pdf:barcode |
| resolution | float | Resolution | pdf:barcode |
