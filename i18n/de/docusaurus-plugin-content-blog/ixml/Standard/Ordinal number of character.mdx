# Ordinal number of character

### Anatomy

```xml
<ord var="var">
    string
</ord>
```

 **Description**:
 ORD returns the ordinal number of a character as an UTF-8 code.

### Content: string

```xml
<content type="string"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | ord |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var |  | N/A |


## Examples

### Example: Example

```xml
<output>
<ord>A</ord>
</output>
<!-- 65 -->
```
