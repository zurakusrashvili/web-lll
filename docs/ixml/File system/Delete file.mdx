# Delete file

### Anatomy

```xml
<file:delete filename="string"/>
```

 **Description**:
 FILE:DELETE deletes a file.

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| filename | string | Filename/URL | file:delete |


## Examples

### Example: Example

```xml
<file:delete filename="./test/ixml.inc"/>
```
