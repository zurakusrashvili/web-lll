# Create

### Anatomy

```xml
<excel:create var="var" format="string"/>
```

**Parents**: excel:workbook

 **Description**:
 EXCEL:CREATE generates an Excel workbook according to a given file format.

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | excel:create |
| format | string | File format | excel:create |
