# Update records

### Anatomy

```xml
<db:update var_data="var" table="string">
    ixml
</db:update>
```

**Parents**: db:connection

 **Description**:
 DB:UPDATE executes a SQL/DML update statement that updates fields of existing rows in a table within the database.

### Content: ixml

```xml
<content type="ixml"/>
```

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var_data | var | Variable name for associated field name and value pairs | db:update |
| table | string | Table name | db:update |
