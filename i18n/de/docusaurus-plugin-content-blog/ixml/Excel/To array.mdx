# To array

### Anatomy

```xml
<excel:array var="var" type="type"/>
```

**Parents**: excel:workbook

 **Description**:
 EXCEL:ARRAY extracts all cell values of a workbook into a multidimensional array.

**Attributes**:

| Name | Type | Description | Defined By |
| ---- | ---- | ----------- | ---------- |
| var | var | Result variable name | excel:array |
| type | type | Evaluation type | excel:array |


**Results**:

| Binding | Type | Predicate |
| ------- | ---- | --------- |
| var | array | no-result-propagation |
